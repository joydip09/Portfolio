# Joydip — Portfolio Website

A clean, dark-themed personal portfolio built with vanilla HTML, CSS, and JavaScript.

## Structure

```
portfolio/
├── index.html        ← Main page
├── style.css         ← All styles
├── script.js         ← All interactivity
├── assets/
│   ├── images/
│   │   ├── weather-app.png
│   │   ├── quiz-app.png
│   │   └── todo-app.png
│   └── icons/
└── README.md
```

## Customization Checklist

- [ ] Replace `Joydip@example.com` with your real email
- [ ] Replace `github.com/Joydip` with your GitHub URL
- [ ] Replace `linkedin.com/in/Joydip` with your LinkedIn URL
- [ ] Add screenshots to `assets/images/`
- [ ] Update Live Demo and GitHub links in each project card
- [ ] Add your photo to `.about-img-placeholder` (or keep the initials)
- [ ] Update skill bar percentages in `index.html` via `data-width` attributes

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flex, animations)
- Vanilla JavaScript (IntersectionObserver, no dependencies)
- Google Fonts: Syne + DM Sans
